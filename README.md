# Comic Generator

Turn a one-line story idea into a single-page noir comic — fully generated by AI.

![Dark Prophecy](output/2026-05-23/story_165242_dark-prophecy_s9y/comic.png)

## How it works

The pipeline runs in three steps:

1. **Plot** — GPT-4o reads your story idea and generates a title, tagline, scene beats, and a dynamic panel layout (4–12 panels, chosen to fit the narrative pacing).
2. **Scenes** — Each scene is expanded in parallel: GPT-4o writes the caption and dialogue, then `gpt-image-1` generates the panel image in the chosen comic style.
3. **Composite** — Pillow stitches all panels into a single greyscale PNG using the AI-chosen layout (weighted rows, splash panels, dense grids — whatever the story calls for).

Output is saved to a timestamped folder so every run is preserved:
```
output/
└── 2026-05-23/
    └── story_165242_dark-prophecy_s9y/
        ├── plot.json      ← full story structure
        ├── scene_01.png
        ├── ...
        └── comic.png      ← the finished page
```

## Setup

**Requirements:** Python 3.9+, an OpenAI API key.

```bash
# 1. Clone and install dependencies
pip install -r requirements.txt

# 2. Create your config
cp config.example.yml config.yml
# Open config.yml and paste your OpenAI API key under openai.api_key
```

## Generate a comic

```bash
python -m src.main --idea "A blind medium receives a photograph of her own future murder"
```

Options:
```
--idea    Your story premise (required)
--style   Comic style to use — matches a file in Styles/<name>.md (default: dylan-dog)
--fun     Switch to absurdist/comedy mode instead of horror/noir
--config  Path to config YAML (default: config.yml)
```

Example in fun mode:
```bash
python -m src.main --idea "A pigeon runs for mayor of New York City and actually wins" --fun
```

## Comic styles

The visual style is defined in `Styles/<name>.md`. Each file contains the image prompt suffix that is appended to every panel generation call. The default style is **Dylan Dog** — Italian noir, black & white pen-and-ink, urban gothic London atmosphere.

To add a new style, create `Styles/my-style.md` and run with `--style my-style`.

## Switching image models

The pipeline is abstracted so you can switch image providers in `config.yml`:

```yaml
providers:
  active_image_provider: dall-e-3   # or: flux (Black Forest Labs Flux Pro)
```

See `Notes/image-model-research.md` for a comparison of available models.
